  <action>
Create `src/app/api/generate/route.ts` as a Next.js App Router Route Handler.

**Directory:** `src/app/api/generate/route.ts` (create the directory if it does not exist)

**POST handler logic (in order):**

1. **Validate env vars** — check `process.env.N8N_WEBHOOK_URL` and `process.env.N8N_WEBHOOK_SECRET` exist; return `{ error: 'Generation service not configured' }` with status 503 if missing.

2. **Auth check** — `const supabase = await createClient()`, get user via `supabase.auth.getUser()`. Return 401 if no user.

3. **Parse and validate request body** — expect JSON `{ brand_id, template_id, image_style_id, idea_text, slide_count }`. Return 400 if any required field is missing.

4. **Atomic credit deduction** — `const admin = createAdminClient()`, call `admin.rpc('consume_credit', { p_user_id: user.id })`. If `!data?.success`, return `{ error: 'Insufficient credits' }` with status 402.

5. **Fetch related data server-side** — fetch brand, template, image_style in parallel using `admin.from(...)`. Return 500 if any lookup fails.

6. **Insert carousel row** — insert into `carousels` with `{ user_id, brand_id, template_id, image_style_id, idea_text, status: 'processing' }`. Return 500 if insert fails.

7. **Fire-and-forget n8n webhook** — call `fetch(N8N_WEBHOOK_URL, { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-Webhook-Secret': N8N_WEBHOOK_SECRET }, body: JSON.stringify(payload) }).catch(err => console.error('[generate] n8n fire failed:', err))`. Do NOT await this call.

8. **Return immediately** — `return Response.json({ carousel_id: carousel.id })` with status 201.

**n8n payload shape** (N8N-02 requirement — full brand object, not just IDs):
```typescript
{
  carousel_id: carousel.id,
  idea_text: body.idea_text,
  slide_count: body.slide_count ?? 7,
  brand: {
    name: brand.name,
    primary_color: brand.primary_color,
    secondary_color: brand.secondary_color,
    voice_guidelines: brand.voice_guidelines,
    product_description: brand.product_description,
    audience_description: brand.audience_description,
    cta_text: brand.cta_text,
  },
  template: {
    id: template.id,
    name: template.name,
    slug: template.slug,
    cover_url: template.cover_url,
    content_url: template.content_url,
    cta_url: template.cta_url,
  },
  image_style: {
    id: imageStyle.id,
    name: imageStyle.name,
    description: imageStyle.description,
  },
}
```

**Anti-patterns to avoid:**
- Do NOT await the n8n fetch — Vercel Hobby has 10s timeout, generation takes 30-90s
- Do NOT call `consume_credit()` twice (no retry logic in POST — retry re-uses existing carousel_id, see Plan 03)
- Do NOT use `createClient()` (user client) for the DB inserts — use `createAdminClient()` for all mutations

<task type="auto">
  <name>Task 2: GET /api/generate/status — poll carousel by id</name>
  <files>src/app/api/generate/status/route.ts</files>
  <action>
Create `src/app/api/generate/status/route.ts` as a Next.js App Router Route Handler.

**Directory:** `src/app/api/generate/status/route.ts` (create the directory)

**GET handler logic:**

1. **Auth check** — `const supabase = await createClient()`, get user. Return 401 (not 404) if unauthenticated — so the client can distinguish auth failure from "job not found".

2. **Read id query param** — `const carouselId = request.nextUrl.searchParams.get('id')`. Return 400 if missing.

3. **Query carousel** — `supabase.from('carousels').select('id, status, slide_urls, post_body').eq('id', carouselId).eq('user_id', user.id).single()`. The `.eq('user_id', user.id)` enforces ownership (belt-and-suspenders alongside RLS).

4. **Return result** — if not found or error: 404. If found: `Response.json({ id, status, slide_urls, post_body })`.

**Import pattern:**
```typescript
import { type NextRequest } from 'next/server'
import { createClient } from '@/lib/supabase/server'
```

Use `export async function GET(request: NextRequest)` — NextRequest gives access to `request.nextUrl.searchParams`.

**Response shape used by Plan 03 polling:**
```typescript
{
  id: string
  status: 'pending' | 'processing' | 'completed' | 'failed'
  slide_urls: string[] | null
  post_body: string | null
}
```
