**Why keep the proxy for Supabase Storage URLs too?**
Supabase Storage public URLs are directly accessible, but routing through our proxy gives us:
1. Consistent `Content-Disposition: attachment` header (triggers browser download instead of opening in tab)
2. Consistent filename control
3. Auth guard (only logged-in users can download)
